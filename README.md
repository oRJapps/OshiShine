# OshiShine
A specialized tool for lighting characters in Fortnite replay videos.


# 👁️ Overview

When recording cinematic shots in Fortnite Replay mode, character lighting can often be challenging due to environmental constraints. OshiShine provides a streamlined workflow to "shine" your favorite characters (Oshi), ensuring they stand out in any scene.

# ✨ Key Features

Character Focus Lighting: Optimized presets to illuminate characters without washing out the environment.

Cinematic Presets: Pre-configured color profiles for various atmospheric needs (e.g., Golden Hour, Spirit Realm, Night Ops).

Standalone Utility: Lightweight executable that works alongside your capture workflow.

Easy Integration: Designed for users who move their captures from Fortnite into Blender or UEFN for further editing.

# 🚀 Getting Started

## Installation

1. Navigate to the Releases page.

2. Download the latest OshiShine_vX.X.exe.

3. No installation required—just run the executable and start shining!

## Usage

1. Launch Fortnite and enter Replay Mode.

2. Run OshiShine.exe.

3. Use the interface to adjust lighting intensity and color temperature to match your character's vibe.
